Transform your monorepo setup from hours to minutes. Our pre-configured templates handle the complexity while you focus on building.

## Getting Started

<Step>
```bash
npx create-turbo-tpl@latest
```

Select "Turborepo" when prompted for your framework choice.
</Step>


## Quick Start Example

<Step copyCommand={false}>
```bash
# Run the CLI
npx create-turbo-tpl@latest

# Choose your setup
✔ Project name: my-turborepo
✔ Select framework › Turborepo
✔ Choose template › turborepo-with-shadcn

# Your monorepo is ready!
cd my-turborepo
pnpm run dev
```
</Step>
