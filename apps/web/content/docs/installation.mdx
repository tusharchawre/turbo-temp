## Quick Start

Bootstrap your entire project with a single command:

<Step>

```bash
npx create-turbo-tpl@latest
```

</Step>

## What Happens Next?

The CLI will guide you through a series of prompts:

- Project name
- Framework preferences
- Template selection

## Example Usage

<Step>

```bash
npx create-turbo-tpl@latest my-todo-list-application
cd my-awesome-project
pnpm run dev
```

</Step>

That's it! Your development environment is ready to go.
