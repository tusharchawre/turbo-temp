## Key Features

- **Real-time Capabilities**: Integrated WebSocket support
- **Prisma ORM**: Shared database client with migrations
- **TypeScript First**: End-to-end type safety

## Quick Start

<Step copyCommand={false}> 
```bash
# Create new project
npx create-turbo-tpl@latest

# Choose configuration
✔ Project name: backend-monorepo
✔ Select framework › Turborepo
✔ Choose template › express-prisma-ws

# Start development
cd backend-monorepo
pnpm install
pnpm run dev:api
```
</Step>
