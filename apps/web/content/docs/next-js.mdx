Transform your Next.js setup from hours to minutes. Our pre-configured templates handle the complexity while you focus on building.

## Getting Started

<Step>
```bash
npx create-turbo-tpl@latest
```

Select "Next.js" when prompted for your framework choice.
</Step>

## Quick Start Example

<Step copyCommand={false}>
```bash
# Run the CLI
npx create-turbo-tpl@latest

# Choose your setup
✔ Project name: my-nextjs-app
✔ Select framework › NextJS
✔ Choose template › NextJs-with-shadcn

# Your project is ready!
cd my-nextjs-app
pnpm run dev
```
</Step>
