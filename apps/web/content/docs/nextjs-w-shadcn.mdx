## Getting Started

<Step>
```bash
npx create-turbo-tpl@latest
```

Select "NextJs" when prompted for your framework choice.
</Step>




## Quick Start

<Step copyCommand={false}>
```bash
# Run the CLI
npx create-turbo-tpl@latest

# Choose your setup
✔ Project name: my-nextjs-app
✔ Select framework › NextJs
✔ Choose template › NextJs-w-shadcn

# Your project is ready!
cd my-nextjs-app
pnpm run dev
```
</Step>
