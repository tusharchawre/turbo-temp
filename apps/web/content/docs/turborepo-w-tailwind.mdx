### Key Features

- **Monorepo Structure**: Efficient management using Turborepo.
- **Shared UI Components**: Shadcn/UI components are reusable across all applications.
- **Type Safety**: Full TypeScript support throughout the project.
- **Consistent Styling**: Tailwind CSS configuration is shared across packages.

## Quick Start

<Step copyCommand={false}>
```bash
# Run the CLI
npx create-turbo-tpl@latest

# Choose your setup
✔ Project name: my-turborepo
✔ Select framework: Turborepo
✔ Choose template: turborepo-with-shadcn

# Your monorepo is ready!
cd my-turborepo
pnpm run dev
```
</Step>

## Project Organization

### Apps Directory
The `apps` directory contains your main applications:

- `web`: The primary web application.
- `docs`: Documentation site (optional).


## Development Workflow

1. Run all apps in development mode:
   ```bash
   pnpm run dev
   ```
2. Build all apps and packages:
   ```bash
   pnpm run build
   ```

## Configuration Files

- **`turbo.json`**: Defines the task pipeline and dependencies.
- **`tailwind.config.ts`**: Shared Tailwind CSS configuration.
- **`tsconfig.json`**: Base TypeScript configuration.
